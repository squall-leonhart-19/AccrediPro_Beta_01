# AccrediPro Universal Course Structure

## Overview

Every certification follows the same 4-tier structure:

| Tier | Name | Price | Modules | Lessons |
|------|------|-------|---------|---------|
| **L1** | Foundation Certification | $97 | 12-21 | 48-84 |
| **L2** | Advanced Clinical | $397 (OTO bundle) | 8 | 64 |
| **L3** | Master Level | $397 (OTO bundle) | 6 | 48 |
| **L4** | Practice Path | $397 (OTO bundle) | 7 | 56 |

---

## Proprietary Methodologies

Each certification has its **own unique framework/method** that runs throughout all content:

| Certification | Methodology | Acronym Meaning |
|---------------|-------------|-----------------|
| **Functional Medicine** | DEPTH™ | Discover, Evaluate, Plan, Treat, Heal |
| **Holistic Nutrition** | NOURISH™ | Nourish, Optimize, Understand, Restore, Integrate, Sustain, Heal |
| **Gut Health** | RESTORE™ | Remove, Evaluate, Support, Target, Optimize, Repair, Establish |
| **Hormone Health** | BALANCE™ | Baseline, Assess, Leverage, Adapt, Nurture, Calibrate, Empower |
| **Stress & Burnout** | ASCEND™ | Assess, Stabilize, Cultivate, Energize, Navigate, Develop |
| **Sleep Health** | REVIVE™ | Reset, Evaluate, Validate, Implement, Verify, Establish |
| **Weight Loss** | TRANSFORM™ | Track, Reset, Assess, Nourish, Sustain, Focus, Optimize, Restore, Maintain |
| **Autoimmune** | RECLAIM™ | Reduce, Evaluate, Calm, Learn, Address, Integrate, Maintain |
| **Mental Wellness** | THRIVE™ | Track, Heal, Regulate, Integrate, Validate, Empower |
| **Longevity** | LIFESPAN™ | Lifestyle, Interventions, Function, Exercise, Sleep, Protect, Assess, Nurture |

### How Methodology Integrates

- **L1 Foundation:** Introduces the methodology framework
- **L2 Advanced Clinical:** Deep application of methodology to complex cases
- **L3 Master Level:** Methodology mastery and pattern recognition
- **L4 Practice Path:** Real-world methodology implementation

### Module Breakdown (Template)
```
Module 0:  Welcome & Orientation
Module 1:  Foundations & Philosophy
Module 2:  Core Concepts
Module 3:  Assessment Framework
Module 4-8: Deep Dive Topics (specialty-specific)
Module 9:  Treatment Planning
Module 10: Lifestyle Medicine
Module 11: Case Studies
Module 12: Certification & Business Launch
```

### Outcome
- Certified [Specialty] Practitioner
- Can start charging clients
