## Avatar Requirements

**Style:**
- Professional headshots
- Warm, approachable smiles
- Age range: 32-55
- Diverse but realistic for US demographics
- High quality but not "stock photo" feeling

**Sources:**
- ThisPersonDoesNotExist.com (AI-generated)
- Licensed stock photos
- Team member photos (with permission)

**File Format:**
- 200x200px minimum
- JPEG or WebP
- Stored in `/public/zombies/`

---

## Usage Rules

1. **Never reuse** - Each profile appears in ONE context (chat OR graduates, not both)
2. **Consistent timing** - Same profiles appear at similar video timestamps each session
3. **Natural spacing** - Don't cluster too many messages together
4. **Mix with real** - Real user messages should appear alongside zombies
5. **Adapt to content** - Messages should relate to what's being said in video

## Database Seeding

```typescript
// prisma/seed-zombies.ts

const ZOMBIE_PROFILES = [
  {
    id: "maria_t",
    name: "Maria T.",
    location: "Austin, TX",
    avatar: "/zombies/maria.jpg",
    tier: "active",
    background: "nurse"
  },
  // ... all profiles
];

// Seed to database
for (const profile of ZOMBIE_PROFILES) {
  await prisma.zombieProfile.upsert({
    where: { id: profile.id },
    update: profile,
    create: profile
  });
}
```
