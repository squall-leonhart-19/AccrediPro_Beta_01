Protocol #1: Gut Healing (5R Framework)

ü¶† 5R Gut Restoration Protocol
1. Remove (Weeks 1-4)
  • Eliminate inflammatory foods: gluten, dairy, sugar, processed foods
  • Address pathogens if present: antimicrobials, anti-fungals as indicated
  • Remove toxins: alcohol, NSAIDs, unnecessary medications
2. Replace (Weeks 1-8)
  • Digestive enzymes with meals (especially if low HCl suspected)
  • Betaine HCl if hypochlorhydria confirmed
  • Bile salts if fat digestion impaired
3. Reinoculate (Weeks 4-12)
  • Broad-spectrum probiotics: 25-100 billion CFU
  • Strain diversity matters‚ÄîLactobacillus + Bifidobacterium species
  • Prebiotic fiber: gradually increase (GOS, FOS, resistant starch)
4. Repair (Throughout)
  • L-glutamine: 5-15g daily
  • Zinc carnosine: 75-150mg daily
  • Collagen peptides: 10-20g daily
  • Aloe vera, DGL, slippery elm as needed
5. Rebalance (Ongoing)
  • Stress management‚Äîgut-brain connection
  • Sleep optimization
  • Mindful eating practices
  • Long-term dietary maintenance

Protocol #2: Immune Support

üõ°Ô∏è Foundational Immune Protocol
Daily Foundation
  • Vitamin D: 2000-5000 IU (target 50-70 ng/mL)
  • Vitamin C: 500-1000mg 2-3x daily
  • Zinc: 15-30mg daily
  • Probiotics: Multi-strain, 25+ billion CFU
Acute Support (First Sign of Illness)
  • Zinc lozenges: Every 2-3 hours (first 48 hours)
  • Vitamin C: Increase to 1g every 2-3 hours (bowel tolerance)
  • Elderberry: 15ml syrup 4x daily
  • Andrographis or Echinacea: As directed on product
Recovery Phase
  • Continue elevated vitamin C for 3-5 days after resolution
  • Probiotics to restore gut flora if antibiotics used
  • Extra rest; immune system depleted post-infection

Protocol #3: Mitochondrial Support

‚ö° Cellular Energy Protocol
Core Mitochondrial Support
  • CoQ10 (Ubiquinol): 100-300mg daily
  • D-Ribose: 5g 2-3x daily
  • L-Carnitine: 1-2g daily (Acetyl-L-Carnitine for brain)
  • Alpha-Lipoic Acid: 300-600mg daily
  • B-Complex: Methylated forms
  • Magnesium: 400-600mg (ATP requires magnesium)
Best Used For
  • Chronic fatigue syndrome / ME
  • Long COVID
  • Post-viral fatigue
  • Aging-related energy decline
  • Statin-induced CoQ10 depletion

Protocol #4: Adrenal/HPA Axis Support

üßò Stress Resilience Protocol
Foundation
  • B-Complex: B5 (pantothenic acid) especially important
  • Vitamin C: 1-2g daily (adrenals concentrate vitamin C)
  • Magnesium: 400-600mg glycinate or threonate
Adaptogens (Choose 1-2)
  • Ashwagandha: 300-600mg standardized extract
  • Rhodiola: 200-400mg (more stimulating)
  • Holy Basil: 300-600mg (calming)
  • Eleuthero: 300-400mg
Sleep/Evening Support
  • Phosphatidylserine: 100-300mg PM (lowers cortisol)
  • L-Theanine: 200-400mg
  • Magnesium: Extra 200-400mg before bed

Protocol Customization Principles

Tailoring to the Individual

  • Start conservative: Especially with sensitive clients
  • Layer in: Don't start everything at once; add 1-2 items weekly
  • Watch for reactions: If new symptoms appear, identify the trigger
  • Adjust based on labs: Use testing to guide, not just symptoms
  • Consider interactions: With medications and between supplements
  • Set duration: Most protocols have an end point; reassess in 3-6 months
