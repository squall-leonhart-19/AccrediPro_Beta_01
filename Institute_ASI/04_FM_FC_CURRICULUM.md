## Program Overview

| Element | Details |
|---------|---------|
| **Credential** | FM-FC™ (Functional Medicine - Foundation Certificate) |
| **Level** | Foundation (Tier 1) |
| **Total Hours** | 25 contact hours |
| **Modules** | 5 core modules |

| **Pass Score** | 70% (35/50) |
| **Timeline** | 4-8 weeks typical |
| **Prerequisite** | None |

---

## Learning Objectives

Upon completion of this program, candidates will be able to:

1. Define functional medicine and distinguish it from conventional approaches
2. Explain the core principles and philosophy of functional medicine
3. Describe the major physiological systems and their interconnections
4. Identify key nutritional and lifestyle factors affecting health
5. Understand the functional medicine framework for assessment
6. Recognize when to refer to qualified practitioners
7. Articulate the pathway to professional certification
