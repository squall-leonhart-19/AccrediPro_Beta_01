## Tier Details

### Tier 1: Foundation Certified (FC™) — $297

**Who it's for:** Career explorers, wellness enthusiasts, career changers

**What's Included:**

| Component | Details |
|-----------|---------|
| Curriculum | 25 hours (5 modules) |
| Dedicated Coach | 24/7 study buddy mode |
| Cohort | 50 classmates (My Circle) |
| Exam | 50 questions, 70% to pass |
| Credential | Digital badge + certificate |
| Directory | Basic listing |
| CE Credits | 6 included |

**Upgrade Path:** → CP™ (save $500)

---

### Tier 2: Certified Professional (CP™) — $1,997

**Who it's for:** Serious practitioners, those building a practice

**Prerequisites:** FC™ or equivalent experience

**What's Included:**

| Component | Details |
|-----------|---------|
| Curriculum | 75 hours (15 modules) |
| Dedicated Coach | Clinical mentor mode |
| Cohort | 30 professionals (My Circle) |
| Case Studies | 10 with coach feedback |
| Live Calls | Monthly group practice calls |
| Exam | 100 questions, 75% to pass |
| Credential | Professional badge + certificate |
| Directory | Featured listing |
| CE Credits | 12 included |

**Upgrade Path:** → BC-™ (save $2,500)

---

### Tier 3: Board Certified Practitioner (BC-™) — $5,997

**Who it's for:** Elite practitioners, practice owners, industry leaders

**Prerequisites:** CP™ required + interview

**What's Included:**

| Component | Details |
|-----------|---------|
| Curriculum | 150 hours (mastery level) |
| Dedicated Coach | Practice advisor mode |
| Human Mentor | 6× 1:1 sessions |
| Cohort | 15 elite (My Circle) |
| Supervised Cases | 10 real client cases |
| Live Calls | Weekly mastermind with Ciro |
| Exam | Comprehensive + case defense |
| Credential | Board badge + wall certificate + pin |
| Directory | Premium + referral priority |
| Board Membership | Automatic (voting rights) |
| CE Credits | 24 included |

**Upgrade Path:** → MC-™ (by application)

---

| Lifetime CE | Unlimited access |
