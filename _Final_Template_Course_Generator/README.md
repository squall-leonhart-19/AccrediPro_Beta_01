## Workflow

### Step 1: Generate Resources
1. Read lesson content
2. Use **01_RESOURCE_GENERATION_PROMPTS.md**
3. Generate 1 handout + 1 tool per lesson
4. Export as PDF

### Step 2: Generate Audio
1. Read lesson content
2. Use **02_AUDIO_SCRIPT_PROMPTS.md**
3. Generate TTS-ready script per lesson
4. Send to Voce/ElevenLabs

### Step 3: Generate Reviews
1. Use **03_REVIEW_GENERATION_PROMPTS.md**
2. Generate 50-100 reviews per course
3. Match to zombie profiles
4. Import to database
